Templates Add-on for Woo OnePage - Lite is a awesome templates Add-on for Woo OnePage Checkout Shop.
== Description ==
Display products with Templates Add-on for Woo OnePage - Lite which is a awesome Add-on for Woo OnePage Checkout Shop.
= Advantages of Templates Add-on for Woo OnePage Add-on - Lite =
* Easy Installation.
* Multiple layout to display products.
* Item's number increment /decrement on the one page.
* Variation swatches for variable products - lite features.
* Ajax based action.
* 100% mobile friendly.

= Required Plugins =
* <strong> [WooCommerce](https://wordpress.org/plugins/woocommerce)</strong>
* <strong> [Woo OnePage Checkout Shop](https://wordpress.org/plugins/woo-onepage)</strong>

== Installation ==

1. Download the plugin woo-onepage-templates.zip file. Extract and upload in your wp-content/plugins folder.
2. Activate the plugin through the 'Plugins' screen in WordPress by clicking 'Activate' button below 'Templates Add-on for Woo OnePage -Lite'.
3.Use the OnePage Shop->Settings screen to configure the your affiliate account tag and country.
4. You are done.
